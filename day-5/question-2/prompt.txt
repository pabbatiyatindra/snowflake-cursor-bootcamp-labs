Create semantic search using embeddings:

3. Find most similar reviews using VECTOR_COSINE_SIMILARITY
4. Build semantic search query for arbitrary text
5. Display similarity scores and rankings

REQUIREMENTS:
1. Use SNOWFLAKE.CORTEX.EMBED_TEXT_768('multilingual-e5-large', text)
2. Create review_embeddings table with embedding_vector column
3. Use VECTOR_COSINE_SIMILARITY(vec1, vec2) for comparison
4. Find top 3 similar reviews to review_id = 1
5. Implement search query for "great product highly recommend"
6. Show similarity scores (0-1 range)
7. ORDER BY similarity DESC, LIMIT results

KEY CONCEPTS:
- Embeddings: Text → 768-dimensional vector (numerical representation)
- Cosine similarity: Angle between vectors (0=different, 1=identical)
- Semantic search: Find similar content without keyword matching
- Caching: Pre-compute and store embeddings

EXAMPLE PATTERN:
CREATE TABLE review_embeddings AS
SELECT review_id, review_text,
  SNOWFLAKE.CORTEX.EMBED_TEXT_768('multilingual-e5-large', review_text)
FROM reviews;

WITH target AS (
  SELECT embedding_vector FROM review_embeddings WHERE review_id = 1
)
SELECT e.review_id, e.review_text,
  VECTOR_COSINE_SIMILARITY(e.embedding_vector, t.embedding_vector) as sim
FROM review_embeddings e
CROSS JOIN target t
WHERE e.review_id != 1
ORDER BY sim DESC LIMIT 3;
